# Multa Ibama
Um script para pegar as mutas do IBAMA, em quatro categorias, do site do governo:
  - Flora 
  - Fauna
  - Pesca
  - Outros
  
Pegando as informacoes:
- Municipio
- Nome / Razao Social
- Valor da Infração
- Data da infração
- Situação do debito 

Gerando Arquivo arquivos CSV, por categoria (Flora / Fauna /Pesca / Outros)
